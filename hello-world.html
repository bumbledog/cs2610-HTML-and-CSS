Unix Command Cheat Sheet

Command Description Example
ls [option] [fileName] List information about the files. $ ls -a
./
../
README.txt
file [option] object_name Classifies a filesystem object. $ file Video.MOV
Video.MOV: ISO Media, Apple QuickTime movie, Apple QuickTime (.MOV/QT)
cd [option] [directory] Change the directory the in which the user is currently working. $ cd ..
mkdir [option] DIRECTORY Create the DIRECTORY. $ mkdir myDirectory
echo [option] [string] Prints the string or variable's value. $ echo Hello!
Hello!
